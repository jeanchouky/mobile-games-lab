# Mobile Games Lab

**21 handcrafted HTML5 games.** No downloads, no ads, no tracking. Pure play, straight from your browser.

Every game is a single HTML file — no frameworks, no dependencies, no build tools. Just open and play.

## Play

Open `deploy/index.html` in any browser, or serve locally:

```bash
cd deploy && python3 -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080)

## Philosophy

- **Single-file games** — every game is one self-contained HTML file
- **Zero dependencies** — no npm, no frameworks, no build step
- **Mobile-first** — touch controls, responsive layout, works everywhere
- **No tracking** — no analytics, no cookies, no data collection
- **Free forever** — no ads, no paywalls, no downloads

## Tech

- Pure HTML, CSS, JavaScript
- Web Audio API for sound
- Canvas and DOM rendering
- Touch and pointer events
- Works in any modern browser
